# Movie
该仓库主要是为了上传自己的电影推荐系统的代码

该系统是爬取猫眼上映的电影数据，并进行简单的数据分析。
使用说明：

该程序包含两个文件：
1 是 2018move 该文件是包含爬取猫眼数据 python 模块。
2 是 express 是数据可视化模块。
下载该项目后，在express 文件夹里，打开终端 输入 npm start
